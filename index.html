Pros y Contras

API OFICIAL

  • mas segura/no baneable
  • mas cara para campañas o recordatorios
  • botones interactivos
  • un numero propio
  • requiere sistema
  • mas estable 24/7

API ALTERNATIVA

  • riesgo de baneo
  • sin costo para campañas o recordatorios
  • respuestas basadas en texto
  • el mismo numero
  • no siempre requiere sistema
  • menos estable 24/7, necesita escaneo de qr
